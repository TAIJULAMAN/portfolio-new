# Portfolio Website

A modern, fast, and responsive personal portfolio website built with React and Vite. Showcasing projects, skills, and providing easy contact options.


## Tech Stack

- **React 19** - UI library
- **Vite 6** - Build tool and dev server
- **Tailwind CSS 4** - Utility-first CSS framework
- **React Router 7** - Client-side routing
- **React Icons** - Icon library

## Project Structure

```
src/
├── pages/              # Page components
│   ├── HomePage/       # Main landing page
│   ├── AboutPage/      # About section
│   ├── ProjectsPage/   # Projects showcase
│   └── ContactsPage/   # Contact form
├── layout/             # Layout wrapper
├── shared/             # Reusable components (Header, Footer, etc.)
└── assets/             # Static assets
```



## Pages

- **Home** - Hero section with introduction
- **About** - Background and skills section
- **Projects** - Portfolio of completed projects
- **Contact** - Contact form for inquiries
